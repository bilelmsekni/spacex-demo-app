# Spacex-demo-app

Ionic Space X demo App


## Setup your machine
-----------------

* Download and install visual studio code:

    https://code.visualstudio.com/

* Install the current version of Node:

    https://nodejs.org/en/

* Install Ionic

    `npm install -g ionic cordova`
